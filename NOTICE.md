# NOTICE — content, affiliation, and your own copy

## This repo ships no game content

`sherlock-agent-eval` is an **evaluation harness** (our original work): the
deterministic Game Master, the scorer, the agent prompts, the case **format**,
and an **original** synthetic toy case written for this repo. It contains **no
text, clues, solutions, transcripts, or case-specific analysis** from any
commercial game.

The method follows the idea/expression distinction in US copyright law: copyright
does **not** protect facts, ideas, systems, or methods, but it **can** protect how
they are expressed. We publish the *method and machinery*; we do not reproduce any
game's protected expression.

**transcribe a case from your own copy** into the case files described in
