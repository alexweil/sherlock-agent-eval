# Map of Ashford-on-Sea

The town is divided into five districts. Every location has an address of the
form `<number> <district-code>`.

| Code | District | What's there |
|---|---|---|
| `N` | North Quay | the harbour, the shipping office, the custom house |
| `S` | South Market | stalls, the market square, the curio trade |
| `E` | East Hill | cottages, lodgings, the lamplighter's round |
| `W` | West Gate | the telegraph office, the coaching inn, the chapel |
| `C` | Centre | lodging houses, the laundries, the courier yards |

To go somewhere you must know its address. Resolve a name or a service into an
address using the directories (`directory-people.md`, `directory-places.md`),
then `./gm visit "<number> <district>"` — for example `./gm visit "2 N"`.

> Visiting costs a clue. Looking things up in the directories is free.
